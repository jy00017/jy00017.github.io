# Regex or Regexp(정규표현식)이란?
특정한 규칙을 가진 문자열의 집합을 표현하는데 사용하는 언어

# 활용 범위
* 프로그래밍 언어
* 텍스트 편집기
* script

# 메타문자
## 메타문자란?
정규표현식을 제어할 수 있는 예약 문자
